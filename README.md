# bluefile

Experimental Rust library for handling X-Midas Bluefiles.

## Usage

Add the following to your project's `Cargo.toml`:

```toml
[dependencies]
bluefile = "*"
```

Examples can be found in the `tests` directory.

## Running Tests

```sh
cargo clippy
cargo test
```
